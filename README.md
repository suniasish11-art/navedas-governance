# Navedas Governance Intelligence Platform

Real-Time AI Order Governance Engine · US/Shopify Aligned · Streamlit Cloud

## Deploy to Streamlit Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect repo → set `app.py` as main file
4. Deploy → upload your CSV via the sidebar

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Dataset
Upload `ecommerce_ production_dataset_5000_rows.csv` via the sidebar file uploader.
